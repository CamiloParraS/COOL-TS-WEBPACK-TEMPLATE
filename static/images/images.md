# Available Images

All images are located in `static/images/` and are in SVG format.

---

## Skew Rectangles

Parallelogram-shaped banner elements with an orange accent strip in the upper-left corner. Available in two colors and two orientations.

| File | Description |
|------|-------------|
| `SkewRectangle_Green.svg` | Green parallelogram skewed to the left (500×100) |
| `SkewRectangle_Green_Flip.svg` | Green parallelogram skewed to the right (500×100) |
| `SkewRectangle_Red.svg` | Red parallelogram skewed to the left (500×100) |
| `SkewRectangle_Red_Flip.svg` | Red parallelogram skewed to the right (500×100) |

**Colors used:** `#00FF80` (green) or `#E60003` (red) for the main body; `#FC8416` (orange) for the accent strip.

---

## Hexagon Shapes

Hexagonal frame elements with a black border ring between two layers of the fill color, creating a bordered hex effect.

| File | Description |
|------|-------------|
| `hex_shape.svg` | Large flat-top hexagon in red (584×507) |
| `hex_shape_orange.svg` | Large flat-top hexagon in orange/amber (584×507) |
| `hex_shape_rotate.svg` | Small pointy-top (rotated) hexagon in red (115×133) |
| `hex_shape_orange_rotate.svg` | Small pointy-top (rotated) hexagon in orange/amber (115×133) |

**Colors used:** `#E60003` (red) or `#fa0` (orange/amber); `#000000` (black) for the border ring.

---

## Warning / Earthquake Banners

Official-style emergency warning cards for earthquake (`GEMPA`) alerts, combining striped patterns with bold typography.

| File | Description |
|------|-------------|
| `warning_gempa_black.svg` | Portrait warning card with red background, diagonal black stripes on top and bottom bands, and black "GEMPA BUMI" text (256×227) |
| `warning_gempa_red_yellow.svg` | Tall portrait warning card with red "GEMPA" heading, "BUMI" subheading, yellow/amber diagonal stripes on top and bottom bands, and an "ALERT" section with a red outline box (264×416) |
| `warning_shape_black.svg` | Hexagonal warning shape with concentric red/black borders, upward and downward red triangles, and red "WARNING" text in the center (584×507) |
