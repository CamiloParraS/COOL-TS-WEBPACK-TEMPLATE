# DS-Digital Fonts

Place your DS-Digital .ttf font files here:

- ds_digital.ttf          (regular)
- ds_digitalbold.ttf      (bold)
- ds_digitalital.ttf      (italic)
- ds_digitalboldital.ttf  (bold italic)

The font can be downloaded (free for personal use) from:
https://www.dafont.com/ds-digital.font

If you skip it, all LCD/clock displays will fall back to system monospace.
